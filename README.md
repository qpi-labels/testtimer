# 충곽 타이머 (Chung-Gwak Timer)

충곽 타이머는 학생들의 효율적인 학습 관리와 동기 부여를 위해 개발된 웹 기반 학습 타이머 플랫폼입니다.

## 🚀 주요 기능

- **학습 타이머**: 과목별 공부 시간을 측정하고 기록합니다.
- **실시간 리더보드**: 다른 사용자들의 학습 시간을 확인하며 동기를 부여받습니다.
- **일일 플래너**: 오늘의 할 일을 관리하고 완료 상태를 체크합니다.
- **상세 통계**: 일간 및 누적 학습 데이터를 시각화하여 학습 패턴을 분석합니다.
- **AI 인사이트**: Google Gemini API를 활용하여 학습 습관에 대한 맞춤형 조언을 제공합니다.
- **유틸리티**: 학술 연구 검색엔진 'Dasein;' 연동 및 QPI API 키 발급 기능을 제공합니다.

## 🛠 기술 스택

- **Frontend**: React 19, Vite, Tailwind CSS 4, Framer Motion
- **Icons**: Lucide React
- **AI**: Google Gemini API
- **Deployment**: Cloudflare Pages / Wrangler
- **Language**: TypeScript

## 📦 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드 및 배포
```bash
# 빌드
npm run build

# 배포
npm run deploy
```
